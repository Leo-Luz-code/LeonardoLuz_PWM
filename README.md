## Execução do Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/Leo-Luz-code/LeonardoLuz_PWM.git
   ```
2. Importe o projeto com a extensão Raspberry Pi Pico W do VS Code.
3. Compile e execute a simulação no diagram.json.
